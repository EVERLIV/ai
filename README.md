# АрендаСити

Аренда и продажа коммерческой недвижимости в Иркутске.

## Stack

- Vite + React + TypeScript
- shadcn/ui + Tailwind CSS
- Supabase (optional, for backend features)

## Development

```sh
npm install
npm run dev
```

App runs on http://localhost:8080

## Build

```sh
npm run build
npm run preview
```

## Test

```sh
npm test
```
